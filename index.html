Sesión 3 INTERFACES DE PROGRAMACIÓN HTML, JS, CSS.

Cuando comencé a trabajar con interfaces de programación, me enfoqué en aprender HTML, JavaScript (JS) y CSS. Estos tres lenguajes son fundamentales para el desarrollo web. HTML (HyperText Markup Language) define la estructura básica de las páginas web. Utilizando etiquetas, puedo crear y organizar contenido como textos, imágenes, y enlaces. CSS (Cascading Style Sheets) se encarga del diseño y la presentación de la web. Con CSS, puedo aplicar estilos como colores, fuentes y márgenes, haciendo que las páginas sean visualmente atractivas y coherentes. JavaScript (JS) añade interactividad a las páginas. Con JS, puedo programar funciones dinámicas, como formularios interactivos, animaciones y la actualización de contenido sin recargar la página. Juntos, HTML, CSS y JavaScript me permiten crear interfaces de usuario ricas y dinámicas, proporcionando una experiencia de usuario atractiva y funcional.

Sesión 4
DISEÑO WEB

Cuando me metí en el diseño web, me di cuenta de que las interfaces y la programación son súper importantes para hacer sitios que se vean bien y funcionen genial. HTML es como el esqueleto de la página, poniendo en su lugar cosas como textos e imágenes. CSS se encarga de darle estilo y hacer que todo se vea bonito y se adapte a diferentes pantallas. JavaScript agrega interactividad, haciendo cosas como que los formularios se actualicen sin recargar la página. Además, usar herramientas como Bootstrap y React hace que todo sea más rápido y fácil. Combinando todo esto, puedo crear sitios web que no solo se ven bien, sino que también son fáciles de usar y súper interactivos.

SESiÓN 6
Fabricación Digital

Cuando aprendí sobre la fabricación digital, me sorprendió lo revolucionaria que es. Esta técnica utiliza tecnologías como la impresión 3D, el corte láser y el fresado CNC para crear objetos físicos a partir de diseños digitales. Primero, se diseña el objeto en un software de CAD (diseño asistido por computadora). Luego, ese diseño se envía a una máquina que lo fabrica capa por capa o lo corta con precisión. La fabricación digital permite prototipar rápido, personalizar productos y producir piezas complejas que serían difíciles o imposibles de hacer con métodos tradicionales. Es increíble ver cómo una idea en la pantalla se convierte en un objeto real en tus manos.
